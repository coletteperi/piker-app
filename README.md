# Piker

A simple, customizable weekly habit tracker. One HTML file, no build step.

**Live site:** _enabled after GitHub Pages turns on (1–2 minutes after first deploy)._

## What it tracks (defaults — all editable)

- **Movement** — a different activity for each day of the week
- **Fruit & Veg** — multiple slots per day (e.g. fruit at breakfast, veg at lunch, veg at dinner)
- **Water** — enter your weight + bottle size, the app computes how many bottles to fill (rule: weight ÷ 2 = ounces/day)
- **Coffee** — counter, up to N cups/day
- **Read 5 Min** — daily yes/no
- **Phone Away @ 9pm** — daily yes/no

## Customize

Tap **Edit** in the top-right of the home screen. You can:

- Rename, delete, or add categories
- Pick a category type: yes/no, counter, multi-slot, water (weight-based), or movement
- Edit per-day activities, slot labels, icons, max counts
- For Water: change weight or bottle size — bottle goal recalculates instantly

All progress saves automatically to your browser (localStorage).

## Run locally

Double-click `index.html`, or open it in any browser. That's it.
